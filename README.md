# Юридическая Компания - Noir Surveillance Style

Премиальный лендинг для юридической компании в стиле "Noir Surveillance" / "Monochrome Cyber".

## Особенности
- **Дизайн**: Строгая черно-белая палитра, OSINT эстетика.
- **Типографика**: Использование кастомного шрифта `TS Block Bold` для заголовков.
- **Анимации**: ShinyText (сияющий текст), эффект сканирования, 3D сетка, параллакс.
- **Адаптивность**: Полная поддержка мобильных устройств.

## Стек
- HTML5
- CSS3 (Custom Properties, Keyframes, Flexbox/Grid)
- Vanilla JS (IntersectionObserver)

## Как запустить
Просто откройте `index.html` в браузере.

## Шрифт
Проект использует шрифт `TS Block Bold` (файл `tsblock.ttf`).
